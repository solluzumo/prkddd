# Практическое задание
## О проекте
Автоматизированная система документооборота на Go предсотавляющая разный доступ к различным функциям системы в зависимости от роли пользователя.

## Основные функции
Пользователя с пометкой преподавателя имеют доступ к загруке документов с различными тегами, просмотром этих документов, а так же получении отсортированных и отфильтрованных документов

Пользователя с пометкой сотрудники могу получать список документов отсортированных по пользователям и не только.

## Стек
GoLang, MongoDB, Redis
